# 🌌 Large Model Suite

[![React](https://img.shields.io/badge/React-19-61DAFB?logo=react&logoColor=black)](https://react.dev/)
[![Tailwind CSS](https://img.shields.io/badge/Tailwind_CSS-4.0-38B2AC?logo=tailwind-css&logoColor=white)](https://tailwindcss.com/)
[![Gemini AI](https://img.shields.io/badge/Google_Gemini-2.5_Flash-4285F4?logo=google-gemini&logoColor=white)](https://ai.google.dev/)
[![License](https://img.shields.io/badge/License-Apache_2.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

A high-performance, multi-modal AI workstation that showcases the three pillars of modern Artificial Intelligence: **Large Language Models (LLM)**, **Large Context Models (LCM)**, and **Large Action Models (LAM)**. Powered by Google's state-of-the-art Gemini 2.5 Flash.

---

## 🚀 The Three Pillars

### 1. 🧠 Large Language Model (LLM)
*Reasoning, Creativity, and Intelligence.*
- **Core Capability**: Deep reasoning, creative writing, and complex problem-solving.
- **Use Case**: Brainstorming architectural designs, philosophical debates, or code generation.
- **Model**: Optimized Gemini 2.5 Flash for low-latency, high-intelligence responses.

### 2. 📚 Large Context Model (LCM)
*Processing Massive Datasets.*
- **Core Capability**: Leverages Gemini's massive context window to "read" and understand huge amounts of data.
- **Use Case**: Paste an entire book chapter, a technical manual, or a long legal document and query specific details instantly.
- **Interface**: Dedicated "Context Library" for managing large-scale inputs.

### 3. ⚡ Large Action Model (LAM)
*Tool Use and Automation.*
- **Core Capability**: Beyond just talking, the model *acts*. It identifies user intent and triggers specific tools.
- **Use Case**: "Check the weather in London and create a task to pack my umbrella."
- **Integrated Tools**: Web Search, Weather API (Simulated), and Task Management.

---

## ✨ Key Features

- **📱 Perfectly Responsive**: A "crafted" UI that feels like a native app on mobile and a powerful workstation on desktop.
- **🎭 Glassmorphism Design**: A sleek, dark-themed interface with blur effects, emerald accents, and fluid animations.
- **🛠️ Real-time Action Logs**: Watch the LAM identify and execute function calls in real-time.
- **📝 Markdown Support**: Beautifully rendered AI responses with support for code blocks, lists, and bold text.
- **⚡ Ultra-Fast**: Powered by Gemini 2.5 Flash for near-instantaneous response times.

## ⚙️ Getting Started

### Prerequisites

- **Node.js**: Version 18 or higher.
- **Gemini API Key**: Get a free key from [Google AI Studio](https://aistudio.google.dev/app/apikey).

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/large-model-suite.git
   cd large-model-suite
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Configure Environment Variables**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

4. **Launch the application**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) in your browser.
